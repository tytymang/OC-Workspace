# 회의록 자동화 가이드

## 회의 준비 (시작)
주인님이 "회의 준비해"라고 하시면 아래 절차를 수행합니다.
1. 브라우저를 실행하여 클로바노트(https://clovanote.naver.com/)에 접속합니다.
2. 좌측 상단의 '새 노트 만들기'(마이크 아이콘) 버튼을 클릭합니다.
3. '음성 녹음 시작' 버튼을 눌러 회의 내용을 녹취하기 시작합니다.

## 회의 종료
주인님이 "회의 종료해"라고 하시면 아래 절차를 수행합니다.
1. 녹음 중인 화면에서 '녹음 종료' 버튼을 클릭합니다.
2. 음성 분석이 완료될 때까지 기다립니다.
3. 생성된 회의록 내용을 추출하여 요약 보고합니다.
4. 요약된 내용을 `memory/YYYY-MM-DD_회의록.md` 파일에 저장합니다.
